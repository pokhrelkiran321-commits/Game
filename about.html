About This Project

Features

  • Supports standard chess rules.
  • GUI interface for easy gameplay.
  • Pieces move according to their specific rules.
  • Includes basic AI for single-player mode.

Credits

  • Created by pnpduy.
  • Tutorial by Ryisnow.
  • Chess pieces images by Ryisnow.

Technology

This project uses CheerpJ 3.0, a WebAssembly-based JVM for the browser, to run the original Java bytecode without any modifications or plugins.

How to Play

  • White's Turn: White always moves first.
  • Move Pieces: Click on a piece to select it, then click on a valid square to move.
  • Special Moves: Supports Castling, En Passant, and Pawn Promotion.
  • Game Over: The game ends upon Checkmate or Stalemate.

Note: If the game doesn't load, ensure your browser is up to date and you are serving this page via a web server (not just opening the file directly).

Project Overview

This project is a fully functional Chess Game implemented in Java. It features a complete graphical user interface and adheres to standard chess rules, including complex moves like castling and en passant.

Experience the game directly on the web, powered by CheerpJ technology which allows Java applications to run seamlessly in modern web browsers without plugins.
